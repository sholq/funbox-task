# FUNBOX - Квалификационное задание для разработчиков HTML/CSS/JS

##Description

Задание выполнено с помощью React, функциональных компонентов, хуков и методологии БЭМ Nested.

Выполнены все требования задания:
1. Фон занимает всю площадь экрана, блок находится по центру вертикально и горизонтально.
2. На планшетах упаковки располагаются треугольником, на смартфонах друг под другом.
3. Информация о продукте может меняться. Карточки адаптированны к переполнению содержимого.
4. Каждая из упаковок может быть выбрана или недоступна для выбора. Выбор осуществляется нажатием на упаковку или на текст «купи» в описании.
5. Можно выбрать одновременно несколько упаковок, а также отменить свой выбор повторным нажатием на упаковку.
6. Состояние наведения применяется к выбранной упаковке не сразу, а после того, как курсор ушел с нее после первоначального выбора. Добавлен интерактивный эффект нажатия на карточку.
